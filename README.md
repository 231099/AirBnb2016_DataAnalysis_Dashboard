# Airbnb 2016 Data Analysis Dashboard (Tableau Project)

##  Project Overview
End-to-end data analysis project on Airbnb 2016 listings, involving data preprocessing in Excel and interactive dashboard development in Tableau.

Explored key business questions related to pricing, availability, and property characteristics, delivering actionable insights through dynamic visualizations and geographic analysis.

---

##  Dashboard Features
- Total cost calculation per listing (Price + Cleaning Fee + Security Deposit)
- Average price analysis across different zip codes
- Revenue trends over time (2016)
- Property type distribution (Apartments, Houses, etc.)
- Price variation based on number of bedrooms
- Geographical visualization of average prices using map
- Interactive filters (Bedrooms, Bathrooms, Zip Code, Listing ID)

---

##  Dataset Description
The dataset is stored in an Excel file and consists of three sheets:

1. **Listings**
   - Contains property details such as price, location, property type, bedrooms, and bathrooms.

2. **Reviews**
   - Includes guest reviews and feedback for each listing.

3. **Calendar**
   - Contains pricing and availability data for listings over time.

---

##  Tools & Technologies
- **Tableau** → Data visualization & dashboard creation  
- **Microsoft Excel** → Data cleaning & preprocessing  

---

##  Key Insights
- Prices vary significantly across zip codes, indicating location is a major pricing factor.
- Apartments and houses dominate the listings compared to other property types.
- Listings with more bedrooms tend to have higher average prices.
- Revenue shows an increasing trend over time in 2016.
- Additional costs (cleaning fees and deposits) significantly impact total price.

---

##  Project Workflow
1. Data cleaning and preprocessing using Excel  
2. Handling missing values and standardizing data  
3. Feature engineering (e.g., total cost calculation)  
4. Data visualization using Tableau  
5. Building an interactive dashboard
